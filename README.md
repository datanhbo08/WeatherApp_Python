# Python Weather App

A desktop application built with **Python** and **PyQt5** that provides real-time weather updates using the **OpenWeatherMap API**. 

## Features
- Fetches current temperature and weather descriptions.
- Dynamic emoji icons based on weather conditions (Storms, Rain, Clear, etc.).
- Comprehensive error handling for invalid city names and network issues.
- Secure API key management using environment variables.

## Tech Stack
- **Language:** Python
- **UI Framework:** PyQt5
- **Data Source:** OpenWeatherMap (REST API)
- **Environment Management:** python-dotenv

## How to Run
Set up your API Key:

Create a .env file in the root directory.

Add your key: OPENWEATHER_API_KEY=your_key_here
